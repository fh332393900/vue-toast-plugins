## How to use
You can install using npm
```bash
npm i vue-toast-pulgins
```
Use this in vue
```js
import vueModalPlugins from './toast/toast'
Vue.use(vueModalPlugins)

this.$toast("没有更多了!!!>_",{duration:3000,type:'success'})
```
